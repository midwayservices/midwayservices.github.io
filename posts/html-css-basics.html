Getting Started with HTML & CSS

HTML (HyperText Markup Language) is the backbone of web pages. CSS (Cascading Style Sheets) makes them beautiful.

Basic HTML Structure

<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

Styling with CSS

You can style HTML with CSS. Example:

body {
  background: lightblue;
  font-family: Arial, sans-serif;
}

With HTML and CSS, you can already create beautiful static websites!
